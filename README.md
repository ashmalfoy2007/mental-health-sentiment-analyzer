# Mental Health Sentiment Analyzer

## 📌 Overview

This project is an AI-based Mental Health Sentiment Analyzer that predicts the emotional state of a user based on text input using NLP and Machine Learning.

## 🎯 Objective

To analyze human emotions from text and provide helpful suggestions for mental well-being.

## ⚙️ Features

* Emotion detection (joy, sadness, anger, fear, love, surprise)
* Suggestion system based on emotion
* Text preprocessing using NLP
* Machine learning classification model

## 🧠 Concepts Used

* Natural Language Processing (NLP)
* Supervised Learning
* TF-IDF Vectorization
* Classification Algorithms

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* NLTK
