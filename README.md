# Mental Health Sentiment Analyzer

## 📌 Overview

This project is an AI-based Mental Health Sentiment Analyzer that predicts emotions from text using Natural Language Processing (NLP) and Machine Learning.

## 🎯 Objective

To analyze user emotions from text and provide meaningful suggestions for mental well-being.

## ⚙️ Features

* Emotion detection (joy, sadness, anger, fear, love)
* Suggestion system based on emotion
* Text preprocessing using NLP
* Machine learning model for classification

## 🧠 Concepts Used

* NLP (Natural Language Processing)
* Supervised Learning
* TF-IDF Vectorization
* Classification Algorithms

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* NLTK

## ▶️ How to Run

1. Install dependencies:
   pip install pandas nltk scikit-learn

2. Run:
   python main.py

## 💡 Example

Input: I feel very stressed
Output: Emotion: fear
Suggestion: Take deep breaths and relax.
